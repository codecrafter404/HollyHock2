# Usage of the Draw Program
The program can only draw pixels to the screen, that means it can't save or load images.
## Keys
### s = Pencil Size
|Key|Usage|
---|---
|=|Rubber|
|Clear|exit|
|x|s = 1|
|y|s = 3|
|^|s = 7|
|/|s = 10|
|(|Black|
|)|Blue|
|,|Green|
|(-)|Red|
|*|Cyan|
|-|Magenta|
|+|Yellow|
|EXE|Switch Pencil Mode Between Circle and Square|
Anything else: Toggle Info view
